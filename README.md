## 📌 專案簡介
這是 **立盈國際 (LIYON Global)** 的官方形象網站，旨在展示企業級 AI 自動化解決方案。網站設計採用 **V1.7 文案架構**，強調「理性、鋒利、結果導向」的品牌調性，協助企業主與高階創作者建立自動化商業護城河。

## 🛠️ 技術棧 (Tech Stack)
*   **HTML5**: 語意化結構。
*   **Tailwind CSS**: 實用優先 (Utility-first) 的 CSS 框架，用於快速構建響應式設計與玻璃擬態 (Glassmorphism) 效果。
*   **Google Fonts**: Inter (英文) + Noto Sans TC (繁體中文).
*   **Open Graph**: 完整的社交媒體預覽標籤 (SEO).
